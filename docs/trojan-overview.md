# 木馬程式概述

木馬程式（Trojan Horse）是一類偽裝成合法或有用軟體、實際上帶有惡意功能的惡意程式。它們本身通常不會自我複製，但一旦使用者啟動，即可能在系統中植入後門、竊取資料、下載更多惡意軟體或控制受害電腦。

## 常見特徵
- 偽裝成免費工具、破解程式或電子郵件附件等誘因。
- 執行時可能要求過多權限或試圖關閉安全防護。
- 使用者常在不知情的情況下同意安裝，或透過社交工程被引誘開啟。

## 防護建議
1. **保持系統與軟體更新**：修補已知漏洞，降低被利用的機會。
2. **僅從可信來源下載**：避免安裝來源不明的執行檔或外掛。
3. **使用安全防護工具**：安裝並更新防毒、防火牆及入侵偵測系統。
4. **警惕社交工程**：慎防釣魚郵件、可疑連結或要求提供敏感資訊的訊息。
5. **定期備份**：備份重要資料，以便在系統受到感染時快速復原。

## 若懷疑遭受感染
- 立即中斷網路連線，防止資料持續外流。
- 使用可靠的防毒或資安工具進行全系統掃描。
- 若無法清除，考慮重新安裝系統並從乾淨備份復原。
- 通報公司或組織的資安團隊，協助調查與防止擴散。

了解木馬程式的行為模式與防範措施，有助於降低受害風險並保護個人與組織資產。
